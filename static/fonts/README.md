# Fuentes Casio opcionales

La interfaz está preparada para cargar las fuentes oficiales de la serie gráfica de Casio desde esta carpeta.

Archivos esperados:

```text
CFX01.ttf
CFX02.ttf
CFX04.ttf
CFX05.ttf
CFX06.ttf
```

Estos archivos no forman parte del proyecto. Para usarlos, descargar el paquete oficial `GraphicSeriesFontSet.zip` desde el sitio educativo de Casio, extraer los `.ttf` y copiarlos en esta carpeta.

Si los archivos no están presentes, el navegador usa fuentes monoespaciadas de respaldo y la aplicación sigue funcionando.
